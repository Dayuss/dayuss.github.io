## Getting Started

### Prerequisites
- Node.js (v18+ recommended)
- npm or yarn

### Installation
1. Clone the repository:
	```sh
	git clone <repo-url>
	cd portov2
	```
2. Install dependencies:
	```sh
	npm install
	# or
	yarn install
	```
3. Run the development server:
	```sh
	npm run dev
	# or
	yarn dev
	```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
